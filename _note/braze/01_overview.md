# Braze

> 개인화된 마케팅 메시지를 보내는 솔루션

- 가장 적합한 유저에게
- 가장 적합한 메시지를
- 가장 적합한 타이밍에
- 가장 적합한 채널로



# Message

- 유저들에 데이터를 수집
- 수집된 데이터를 바탕으로 마케팅 메시지를 보낸다.



# 수집 데이터

- Custom Attribute
  - 유저의 상태를 나타내는 속성 값
    - 이름, 핸드폰번호
    - 마케팅 수신동의 여부 등등....
  - 브레이즈에서 제공하는 data type
    - string
- Event Property
  - 유저가 취하는 행동
  - 구매를 한다거나, 특정 버튼을 클릭한다거나, 플로팅배너를 봤다던가..
