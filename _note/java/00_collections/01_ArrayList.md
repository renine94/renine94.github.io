# Collections - ArrayList

```java
import java.util.*;

public class HelloWorld {

     public static void main(String[] args){
        ArrayList<String> arr = new ArrayList<String>();
        arr.add("one");
        arr.add("two");
        arr.add("three");
    
        for (String item: arr) {
            System.out.println(item);
        }
     }
}
```
