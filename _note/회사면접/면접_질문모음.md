**array 와 list 차이**

  - array
    - 인덱스 임의접근 빠름, 메모리공간 연속적할당되있어 효율적
    - 크키가 고정되어있어 변경어렵, 삽입, 삭제시 데이터 이동해야해서 비효율적
  - list
    - 크기 동적 변함, 데이터 추가/삭제 용이
    - 인덱스 접근시 순차탐색 필요해서 느리다, 메모리 공간이 비연속적으로 할당되어있어 비효
