str ==> byte : `encode`

byte ==> str : `decode`

```python
def to_str(value: Union[str, bytes]) -> str:
  if isinstance(value, bytes):
    return value.decode('utf-8')
  return value


def to_bytes(value: Union[str, bytes]) -> bytes:
  if isinstance(value, str):
    return value.encode()
  return value
```
